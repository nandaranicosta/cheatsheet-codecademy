Basic HTML Tags

Tag Name Description
<p> Paragraph Defines a paragraph
<div> Division Divides off a section of code
<a> Anchor Used to link to other locations
<ul> Unordered List Lists elements in Bullet point
<ol> Ordered List Lists elements in numbers or some ordered way
<span> Span To apply some CSS to a small part of text
<em> Emphasize Makes text italicized
<strong> Strong Makes text bold
<br> Break Breaks to the next line

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows definig the column headers in the table
<tbody> Table Body The set of rows containing the actual table data.
<tr> Row Defines an entire row of the table
<th> Head Usually the column heads in a table. Bold.
<td> Table data Cell Contents
<tfoot> Footer Contains all the footer of the table

Forms Tags

Tag Name Description
<form> Form Wrapper element for all forms
<label> Label To properly identify an input
<input> Input Takes the value of the attribute
<textarea> Input To accept a multiple line input

Semantic Tags

Tag Name Description
<header> Header Defines a header
<footer> Footer Defines a footer
<nav> Navigation Defines a menu
<main> Main Define the content of the page
<section> Section Content with the same theme
<article> Article Content that makes sense on its own
<figure> Figure Encapsulates media
<figcaption Caption Inside <figure>, defines the caption of the image
